Mini Chat – Interface & code

Présentation du projet

Ce projet consiste en la création d’un mini chat interactif développé à l’aide de Visual Studio Code. Il permet aux utilisateurs d’échanger des messages via une interface simple et intuitive.

L’objectif était de mettre en pratique les bases du développement web, notamment la structuration du code, la gestion des interactions utilisateur et l’affichage dynamique des messages.

  • Type : Application web
  • Fonctionnalités : Envoi et affichage de messages
  • Langages : HTML / CSS / JavaScript
  • Outil : Visual Studio Code
  • Objectif : Comprendre la logique et l’interactivité
